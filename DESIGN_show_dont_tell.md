# Show, don't tell — Eyes on You 디자인 원칙

> 글로 명시 설명 < 플레이어가 모험·체험으로 자연스럽게 알아가는 것.
> 사용자 결정 (2026-05-04). 모든 텍스트/연출 의사결정의 상위 기준.

---

## 핵심 원칙

1. **체득 우선** — 플레이어는 적의 행동을 글이 아닌 첫 조우의 눈으로 배운다.
2. **간섭 최소** — VEIL 코멘트는 "지시"가 아니라 "반응". 임무 외에는 거의 침묵.
3. **여백 두기** — 침묵·느린 컷·빈 화면이 글보다 더 많은 것을 전달한다.
4. **단서 집중** — 정보가 필요할 땐 한 단어, 한 색깔, 한 그림자만.

---

## 적용 규칙

### 텍스트 분량
- 도감 카드: 3줄 이내. 행동 단서만, 공략 언급 금지.
- VEIL 자막: 1~2줄. 한 호흡에 들어가야 함.
- 브리핑: 1줄 + 시스템 텍스트 1단락. 의뢰인/배경 설명은 뺀다.
- 결말: 4종 모두 5줄 이내. 길게 쓰지 말고 시각·사운드·공백으로.

### 강조
- 적의 행동 단서(LED, 조준선, 그림자 등)는 글로 풀지 말고 **단어 한 개를 노란 강조색**으로.
- "이렇게 하세요"는 금지. "이런 일이 일어났어요"까지만.

### VEIL 톤
- 조언이 아닌 반응. "조심해요" < "...두 명이에요."
- 임무 외 발화는 한 호흡씩. 길게 설명하면 캐릭터가 약해진다.
- 보스 페이즈 같은 게임 시스템 알림도 "패턴이 바뀌었어요" 같은 짧은 코멘트로.

### 튜토리얼
- 글이 아닌 환경이 가르친다. 적이 막은 길, 발판이 부족한 갭, 잠긴 문이 다음 행동을 유도해야 함.
- (현재 5단계 글 기반 → 향후 환경 기반으로 점진 이주 후보)

### 결말
- 결말 4종은 텍스트 위주 → 시각/음향/공백으로. 결말 D(유령 임무)는 이미 침묵 활용 — 다른 결말도 그 방향.

## 적용 후보 목록 (점진 이주)

| 영역 | 현재 상태 | 목표 |
|---|---|---|
| 도감 카드 | blurb + tactic 이중 | blurb 한 줄 + 키워드 강조 (✅ 완료) |
| 오프닝 VEIL | 3줄 인사 | 2줄 짧게 (✅ 완료) |
| 보스 페이즈 알림 | 큰 영문 배너 | VEIL 짧은 자막 (✅ 완료) |
| 브리핑 시스템 텍스트 | 의뢰인/지원/배경 단락 | "OPERATION X — 한 줄 목표" (✅ 완료) |
| 튜토리얼 5단계 | 화면 글 안내 | 환경 유도 (⏳ 향후) |
| 결말 4종 | 텍스트 비중 큼 | 침묵·시각·사운드 (⏳ 향후) |
| 레벨업 카드 | 카드 + VEIL 코멘트 | 카드만, VEIL 톤 다운 (⏳ 향후) |
| 도감 외 적 행동 학습 | 텍스트로 설명 | 첫 조우 시 잠깐 strobe + 행동 1회 미리보기 (⏳ 향후) |

## 무엇을 쓰지 말 것

- "→" 기호 뒤에 공략 풀이.
- "이렇게 하세요" / "X가 도움돼요" / "~게 좋아요" 류 instructional 톤.
- 캐릭터(VEIL) 입을 빌린 게임 시스템 안내. ("새 스킬을 골랐어요!" 같은)
- "위험도 ★★★, 이 구역은 어려워요" — 위험도 표시는 별 자체로 충분.

---

## 무엇을 더 쓰면 좋을지

- 환경 단서(잠긴 문, 깜빡이는 LED, 멀리 들리는 소리)로 호기심 유발.
- VEIL의 침묵·말 끊김·"...".
- 적의 첫 등장 1~2초 멈춤 + 행동 한 번 펼쳐 보이기.
