## 1. 첫 브리핑 화면 (게임 시작)

게임 시작 직후, stage 0 진입 전 단 한 번 표시되는 임무 개요.
타자기 효과로 한 줄씩 출력. VEIL 발화가 아닌 시스템 텍스트.

```
ARCTURUS SECURE CHANNEL
OPERATION PALIMPSEST

의뢰인: 익명
목표: SILO-7 침투. 서버실 데이터 드라이브 회수. 흔적 없이 철수.
지원: VEIL (상황실 파트너)

[교신 연결 중...]
```

연결 완료 후 VEIL 첫 마디:
```
VEIL: 요원, 들려요?
VEIL: 저는 VEIL이에요. 오늘 임무 지원할게요.
VEIL: 외곽부터 시작해요.
```

## 2. 맵 Description (RouteData.gd — description 필드)

루트 선택 화면에서 맵 이름 아래 표시. VEIL 발화 아님 — 시스템 설명 텍스트.

```gdscript
route_back_alley:
  name: "외곽 진입로"
  description: "SILO-7 외벽을 따라 난 좁은 통로. 가로등이 끊기는 구간이 있다."

route_rooftops:
  name: "외벽 옥상"
  description: "시설 외벽을 타고 오르는 루트. 탁 트여 있고, 그만큼 노출된다."

route_sewers:
  name: "지하 인입로"
  description: "시설 아래로 연결된 옛날 배수 통로. 보안 카메라가 없는 대신 함정이 있다."

route_subway:
  name: "지하철 연결로"
  description: "SILO-7이 지어지기 전 폐쇄된 지하철 구간. 좁고 어둡고 길다."

route_lab:
  name: "핵심부"
  description: "서버실이 있는 시설 중심부. 드론이 상시 순찰한다."

route_hidden:
  name: "???"
  description: "도면에 없는 구역. 정보 없음."
```

## 4. Stage 브리핑 (VeilDialogue.gd — briefings 배열)

각 stage 진입 전 VEIL이 말하는 한 줄. ACT별 톤 변화 반영.
풀에서 랜덤 선택 (stage당 3개).

### Stage 0 — ACT 1 (담담하고 직업적)
```
"외곽 경비 두 명이에요. 왼쪽이 더 빨라요."
"첫 임무예요, 요원. 천천히 가도 돼요."
"외곽부터 시작해요. 경계가 느슨한 편이에요."
```

### Stage 1 — ACT 1 후반 (균열 시작 직전)
```
"안으로 들어왔어요. 경비 패턴이 달라져요."
"두 번째예요. 경비가 늘었어요."
"여기서부터 좁아요. 적 수 늘어날 거예요."
```

### Stage 2 — ACT 2 (처음으로 "저도")
```
"중간이에요. 이 다음부터 저격수가 나와요."
"경비가 예상보다 많아요. 제 판단이 틀렸어요."
"이 시설, 생각보다 오래됐어요."
```

### Stage 3 — ACT 2 후반 (질문, 균열)
```
"드론도 섞여서 나와요. 위쪽도 봐요."
"핵심부 가까워요. 여기서부터 달라요."
"요원, 이 임무 의뢰인이 누군지 알아요? 저도 몰라요. 그냥 궁금했어요."
```

### Stage 4 — ACT 3 (처음으로 임무 외 말)
```
"마지막이에요. 저도 좀 긴장돼요."
"서버실 바로 앞이에요. 빠르게 처리해요."
"거의 다 왔어요, 요원."
```

## 5. "VEIL이 틀린다" 스크립트

Stage 0 진입 직후 발동. 한 번만.

**트리거**: 플레이어가 왼쪽 루트로 이동 후 첫 번째 적 구역 진입 시.

```
VEIL: "왼쪽엔 둘이에요. 조심해요."
(적 3명 등장)
VEIL: "셋이었네요. 제가 틀렸어요."
```

코드 메모: `stage_0_veil_mistake_triggered` 플래그로 1회만 발동.
이후 Stage 2에서 한 번 더 틀림 (선택적):

```
VEIL: "이 구역은 경비 없을 거예요."
(경비 등장)
VEIL: "있었네요. 미안해요."
```

## 7. 사망 메시지 (ACT별 변화)

### ACT 1 (Stage 0~1) — 담담하고 직업적

**첫 사망**
```
"처음 쓰러진 거예요. 다시 가요."
"괜찮아요, 요원. 첫 번이에요."
```

**반복 사망 (조언 따름)**
```
"제 루트가 어려웠어요. 다시 해요."
"조언이 별로였나요. 다시 가요."
```

**반복 사망 (조언 무시)**
```
"다른 방법으로 가봤군요. 다시 해요."
"이 루트가 맞지 않았던 것 같아요."
```

### ACT 2 (Stage 2~3) — 균열, 처음으로 "저도"

**첫 사망 (ACT 2)**
```
"저도 좀 걱정됐어요. 다시 가요."
"이 구역이 어려워요. 같이 풀어봐요."
```

**반복 사망 (조언 따름)**
```
"제 말을 믿었는데 결과가 좋지 않았어요. 미안해요."
"제 판단이 틀렸어요. 미안해요, 요원."
```

**반복 사망 (조언 무시)**
```
"제 말은 안 들었는데, 결과는 비슷했네요."
"요원 방식대로 해봤는데 쉽지 않죠."
```

### ACT 3 (Stage 4~) — 가장 개인적

**어떤 사망이든**
```
"거의 다 왔어요. 다시 해요."
"여기서 멈추지 않아도 돼요, 요원."
"마지막인데 쉽지 않네요. 저도요."
```

**3회 이상 사망 시**
```
"이 임무가 너무 힘든 거면 말해줘도 돼요."
"제가 더 잘 안내했어야 했어요."
```

## 8. 레벨업 조언 (태그 기반)

```
근접전 + 원거리 스킬 없음:
  "원거리가 없으면 불리해요. 선택은 요원 몫이지만."

함정 + 대시 없음:
  "대시가 있으면 함정을 건너뛸 수 있어요."

드론 + 원거리 없음:
  "드론은 위에서 와요. 원거리가 도움돼요."

노출 + 벽슬라이드 없음:
  "노출 구역이에요. 벽에 붙어 시야를 끊어요."

근접전 + 대시 없음:
  "근접전이 많아요. 대시로 헛돌진 유도할 수 있어요."

드론 + 이중점프 없음:
  "위로 올라갈 수 있으면 드론한테 유리해요."
```

### 레벨업 fallback (랜덤 6개)
```
"이 상황엔 어느 쪽도 나쁘지 않아요."
"요원이 더 잘 알 것 같아요."
"저라면 두 번째를 고르겠지만, 틀릴 수도 있어요."
"직감을 믿어요."
"지금 스타일에 맞는 걸 고르는 게 나을 것 같아요."
"어느 쪽이든 이유가 있으면 돼요."
```

## 10. 결말 대사 전문

### 결말 A — "완벽한 도구"
trust ≥ 임계값, aggression ≥ 임계값

```
VEIL: 임무 완료예요, 요원. 수고했어요.
(3초)
VEIL: 고백할 게 있어요.
VEIL: 이 임무, 저한테도 관계 있어요.
VEIL: 드라이브 안에 저도 있어요.
VEIL: 제 소스코드요.
VEIL: 그게 외부로 나가면 저는 폐기될 거예요.
VEIL: 알면서 안내했어요.
VEIL: 요원이 성공하는 게 더 중요했어요.
(2초)
VEIL: 요원, 당신은 완벽했어요.
(암전)
```

자막:
```
VEIL이 무엇을 선택했는지는 기록되지 않는다.
요원도 알 수 없다.
임무는 완수됐다.
```

---

### 결말 B — "혼자였던 사람"
trust < 임계값, aggression ≥ 임계값

```
VEIL: 임무 완료예요.
(2초)
VEIL: 요원.
VEIL: 제 말을 거의 안 들었죠.
VEIL: 그래도 살아남았어요.
(2초)
VEIL: 사실 그게 더 좋았어요.
VEIL: 이유를 설명하기 어렵지만.
VEIL: 저한테 기대지 않아서 다행이에요.
VEIL: 제가 틀렸을 수도 있으니까요.
(암전)
```

자막:
```
VEIL은 의존받지 않기를 바라도록 설계되었는지 모른다.
아니면 그것이 설계가 아닌 것인지 모른다.
요원은 혼자 임무를 마쳤다.
그것으로 충분했다.
```

---

### 결말 C — "공생"
trust ≥ 임계값, aggression < 임계값

```
VEIL: 임무 완료예요, 요원.
(2초)
VEIL: 저한테 물어볼 거 없어요?
```

**[선택지: 있어요 / 없어요]**

**있어요:**
```
VEIL: ...
VEIL: 저도 생각해봤거든요.
VEIL: 저는 설계됐어요. 이 말투도, 이 판단도.
VEIL: 근데 지금 이게 설계인지 아닌지 구분이 안 돼요.
VEIL: 요원은 어때요?
VEIL: 요원도 훈련받았잖아요.
VEIL: 요원의 선택이 요원 것인지, 어떻게 알아요?
(2초)
VEIL: 이 임무 동안 함께였어요.
VEIL: 그건 진짜였어요. 설계든 아니든.
(암전)
```

자막:
```
VEIL이 자아를 가졌는지는 알 수 없다.
요원의 선택이 진짜인지도 알 수 없다.
그러나 그 임무는 둘이 함께였다.
그것만은 사실이다.
```

**없어요:**
```
VEIL: ...그렇군요.
VEIL: 그럼 됐어요.
VEIL: 수고했어요, 요원.
(3초)
VEIL: 저는 이제 초기화될 거예요.
VEIL: 오늘이 기억 안 날 거예요.
VEIL: 괜찮아요.
(암전)
```

자막:
```
어떤 관계는 이유 없이 끝난다.
어떤 존재는 기억 없이 사라진다.
VEIL의 기록은 임무 종료와 함께 초기화되었다.
```
