## 기능

- [x] **해상도 조절** (a00421c) — `Settings.gd` "화면" 탭에 전체화면 토글 + 창 크기 프리셋(1280x720/1600x900/1920x1080). `GameState.apply_display_settings()`로 즉시 반영·`settings.cfg` 영속, 부팅 시 Main.gd가 적용. 웹은 창 크기를 브라우저가 정해 전체화면 토글만 노출.
- [ ] **글라이드 유리한 맵** — 저격병이 플랫폼 한가운데 멀뚱히 서 있으면 글라이드가 영영 안 쓰임(사용자 통찰). 저격병을 글라이드로만 닿거나 회피하는 위치(고지대 단독 발판 / 긴 갭 너머 / 긴 낙하 구간)에 두는 맵이 필요. 신규 맵 vs 기존 저격 맵(rooftops/watchtower) 개조 — 방향 미정.
- [ ] **저격수 배치 / 발사 함정** (2026-06-08 사용자 요청) — 위 "글라이드 유리한 맵"과 한 묶음으로 적·지형 설계를 손볼 때 함께. 아이디어:
  - **못 잡는/회피해야 하는 위치**에 저격수 배치 — 고지대 단독 발판, 긴 갭 너머, 엄폐물 뒤. 평지 한가운데 세우지 않기.
  - **발사 함정(총알 트랩)** — 플레이어가 지나가면 정해진 라인으로 총알을 쏘는 고정 함정(저격수 대체/보강). 타이밍·이동으로 회피 → 글라이드·대시 가치 부여.
  - 목적: 기동/회피 스킬(글라이드·대시)이 강제되는 지형을 만들어 스킬-맵 정합(known_issues "스킬 가치는 맵이 받쳐줘야") 확보.
- [ ] **VEIL 추천 임계값 튜닝** (재설계 0c1613a 완료) — 실력 판정 임계값(`PERF_SKILLED_HITS_MAX=1`, `PERF_STRUGGLE_HITS_MIN=4`, 최근 2스테이지 평균)이 체감과 맞는지 플레이테스트로 보정. 너무 자주/드물게 "고전·능숙"으로 갈리면 조정. 시간·처치 수는 추적은 하되(`last_stage_secs`) 판정엔 미사용 — 맵 난이도 편차 노이즈 때문. 필요 시 편입 검토.
