## 결정 사항
- **6개 맵 ↔ SILO-7 매핑**: 도시 다양한 장소가 아니라 SILO-7 안의 진입 경로들로 재정의 (외곽→옥상→지하→지하철→핵심부→격리 서버실). FULL_STORY의 단일 임무 컨셉(OPERATION PALIMPSEST)과 정합.
- **잠긴 문 vs ???루트 분리**: 잠긴 문은 Stage 3에 시각적 복선만(콜리전 없음, 진행 안 막음). ??? 진입은 Stage 4 루트 선택에서 별개로. 게임이 ??? 존재를 천천히 암시하게.
- **VEIL-1/2 표시**: 화면 하단 자막 + 발화자 태그 색 구분(VEIL-1 빨강, VEIL-2 노랑, VEIL 청록). 미니멀 벡터 톤과 일관.
- **HP 5→3**: 기본 5는 사실상 무한이라 위협이 없음. 3으로 줄이면 돌진/폭탄/저격 한두 발에도 위기감.
- **shield = 죽기 직전 부활**: 기존 "2뎀 이상을 1로" 룰은 적이 대부분 1뎀이라 사실상 무의미했음. 라이프라인 형태가 직관적이고 가치 명확.
- **wall_slide = 공중 글라이드**: 게임에 가운데 벽이 없어서 "벽타기" 의미 없음. 글라이드는 옥상 같은 높이차 큰 맵에서 안전 이동에 사용 — 효과만 변경하고 id 유지(코드 영향 최소).
- **regen on_stage_clear heal 제거**: 매 stage 풀 회복이 의도된 동작이라(부스 단발성), 클리어 시 +1은 중복. regen은 max_hp +1 즉시 효과만 남김.
