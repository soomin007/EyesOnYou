## 결정 사항
- **AI 컨셉 유지** — 외부 픽셀/벡터 자산 거부. 코드 도형 + 외곽선/디테일이 정체성. *Why:* 사용자 명시 "AI가 만들었다는 컨셉을 밀려면 네가 만드는 게 제일 낫다". *How:* 새 비주얼은 _filled/_filled_circle 헬퍼로
- **5두신** — 4두신 chibi / 8두신 사실적. 5가 게임 캐릭터다움 균형. *Why:* 사용자 "다리가 길고 상체가 없는 수준" 피드백
- **신뢰도 = trust − aggression** — 둘 다 높으면 neutral, 한쪽 우세일 때만 명확. *Why:* 둘이 독립적으로 결말에 영향, 단계는 차이로
- **prefix random 풀** — 같은 단계에서도 hover마다 다름. *Why:* 사용자 "다 똑같이 음… 이렇게 나와서 단조롭다"
- **가시 dmg 옵션** — sewers 우측만 dmg 2로 차등. 다른 함정 영향 없음
- **HUD 2줄 + 스킬 작게** — 한 줄에 다 넣으면 화면 밖. 스킬은 부차 정보. *How:* hb(18px) + hb2(14px)

### 결정 사항
- **VEIL 멘트가 추천 source** — trust/aggression 헤드로 추천하지 않음. *Why:* 사용자 "추천 표시가 가리키는 것과 멘트가 다르다". *How:* advice.family 단일 source → ★ 일치
- **튜토리얼 SKILL 단계 항상 노출** — 액티브/패시브 모두. *Why:* 패시브여도 "이 스킬 받았다"를 명확히 보여줘야 함. 액티브는 키 학습. *How:* sign 형태만 분기 (키캡 vs 텍스트), 더미 처리는 동일
- **냉각 비밀 통로** — 더블점프 + 외곽 발판 (폭 120). *Why:* 메인 spine 단축으로 줄어든 진행감을 탐험 보상으로 메움. 단조로움 + "왜 올라가야?" 둘 다 해결
- **가시 파이프 베이스** — 30px 빨간 띠 → 10px 어두운 금속. *Why:* sewers 우측 등 mid-air 가시가 base 없이 떠보였음. 파이프 형태가 "박혀 있다" 인상
