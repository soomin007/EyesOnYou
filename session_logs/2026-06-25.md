## 결정 사항
- **신뢰 임계 2/3(데모):** 과반 초과/2-3/4 중 "확실한 다수 2/3". 한 런 추천 5~6회 중 4번 이상 따라야 신뢰.
  공격축 임계(`SCORE_THRESHOLD=4`)는 피드백이 신뢰축만 지적해 유지. 데모 콘텐츠 동결 예외(밸런스 결함 수정).
- **본편(EyesOnYou-Full)엔 임계 미적용:** 본편은 막3+엔딩 재설계로 EndingResolver를 통째 교체할 예정.
  이번 동기화는 패치노트 UI·키설정 안내 한정.
- **키 복원 보고 = 코드 아님:** 복원 로직은 처음부터 save_settings 호출 + 헤드리스 라운드트립 PASS로 정상.
  GitHub Actions(deploy.yml) 자동 배포라 06-24 키 수정도 이미 배포됨 → 브라우저 캐시/IndexedDB 옛 cfg가 원인.
  캐시 안내 + known_issues로 종결.
- **패치노트 UI = STATE_MODE 패턴 재사용:** 단일 소스는 GameInfo.PATCH_NOTES, 항목 추가만 하면 UI 자동 반영.
